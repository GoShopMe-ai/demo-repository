# Acceptance Criteria
## GoShopMe MVP

## 1. AI Chat

AI responds to:

Text-based queries

Image-based search

Video-based search

AI maintains context across messages

AI produces personalized recommendations

## 2. User Profile

User can:

Add and edit full address details

Upload a profile picture

Manage notification preferences

Manage payment methods

Email and phone are verified during registration

## 3. Marketplace Flow

Product catalog displays:

Description

Images

Items can be added to a unified cart

Orders can be tracked

Returns can be initiated only for integrated merchants

## 4. Affiliate Flow

Affiliate flow triggers only when marketplace inventory is unavailable

User is clearly informed before redirection

Returns for affiliate purchases are not handled by GoShopMe

## 5. Subscription

User can:

Start a free trial

View current subscription

Upgrade subscription

Access is restricted without an active subscription

## 6. Stability & UX

Platform works on mobile web

Chat supports text, image, and video inputs

No critical flow breaks during AI interactions
