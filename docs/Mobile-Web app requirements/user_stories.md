# User Stories
## GoShopMe

## Shopper – Onboarding & Discovery

As a user, I want to complete a style quiz so the AI understands my preferences.

As a user, I want the AI to consider my quiz answers (styles, sizes, colors, brands).

As a user, I want the AI to consider my country so recommendations match the current season.

As a user, I want to describe what I need in natural language instead of using filters.

## Shopper – Shopping & Purchase

As a user, I want to manage items via chat.

As a user, I want a unified cart across merchants.

As a user, I want my payment methods stored securely.

## Shopper – Orders & Returns

As a user, I want to track all marketplace orders in one place.

As a user, I want to initiate returns via GoShopMe for integrated merchants.

As a user, I understand that returns for externally purchased items must be handled with the merchant.

## Shopper – Affiliate Fallback

As a user, I want to be able to find items even if they are not in the marketplace.

As a user, I want to be clearly informed when I am redirected outside GoShopMe.

## Platform & Creators

As the platform, I want to learn from user interactions to improve AI accuracy.

As a creator, I want to onboard and curate collections that enhance discovery.
