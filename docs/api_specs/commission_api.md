# Commission & Earnings API
## (Two-Sided Model)

## User & Creator Earnings (Paid by GoShopMe)

### GET /earnings/me

Returns:

Pending

Confirmed

Paid

Balance

### GET /earnings/me/history

## Platform Commission Revenue (Admin / Finance)

### GET /admin/commissions/platform

Returns commission earned from merchants.

### GET /admin/commissions/users

Returns commissions paid to users.

### GET /admin/commissions/creators

Returns commissions paid to creators.

**Note:** Returned orders automatically cancel earnings.
