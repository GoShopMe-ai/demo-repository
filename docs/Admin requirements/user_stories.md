# Admin User Stories
## GoShopMe Admin Application

## Reporting & Analytics

As a Finance Admin, I want to see revenue split by stream so I can track performance.

As a Finance Admin, I want to see free trial conversion rates.

As a Growth Admin, I want to see onboarding channels to optimize acquisition spend.

As an Admin, I want to export reports for further analysis.

As an Admin, I want to see active user metrics to measure engagement.
